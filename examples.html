Question 2

x = 5
y = 10
x, y = y, x

Question 3

What will this code output?

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Error!")
finally:
    print("Done")
  • Error!
  • Done
  • Error!
    Done
  • No output

Answer: Error! followed by Done (on separate lines)

Question 5

What will this list comprehension produce?

result = [x**2 for x in range(5) if x % 2 == 0]
print(result)
  • [0, 1, 4, 9, 16]
  • [0, 4, 16]
  • [0, 2, 4]
  • [1, 9]

Answer: [0, 4, 16]
